# 3D RPG Project

Unity로 만든 간단한 3D RPG 게임 프로젝트입니다.

---

## 프로젝트 개요

이 프로젝트는 플레이어와 적(Enemy)이 상호작용하며 전투를 벌이는 기본적인 RPG 시스템을 구현했습니다.  
적을 공격하고, 플레이어가 경험치(EXP)와 골드를 획득하며 성장하는 시스템을 포함하고 있습니다.

---

## 주요 기능

- **플레이어(Player)**
  - 이동과 공격 상태 구현
  - 적을 자동으로 탐색 및 공격
  - 레벨업 및 스탯 증가 시스템
  - 골드와 다이아몬드 수집
  - 체력(HP), 마나(MP), 경험치(EXP) UI 표시 및 갱신

- **적(Enemy)**
  - 체력과 방어력, 공격력 데이터 관리
  - 플레이어 감지 및 공격
  - 공격 쿨타임 처리

- **스폰 시스템**
  - `EnemySpawner`와 `PlayerSpawner`로 적과 플레이어 위치 관리 및 적 소환

- **버프(Buff) 시스템**
  - 공격력, 체력, 방어력 버프 및 타이머 UI 표시

- **퀘스트(Quest)**
  - 몬스터 처치 퀘스트 구현
  - 퀘스트 진행도 UI 표시

- **스탯 업그레이드**
  - 골드를 사용해 공격력, 체력, 방어력 레벨업

- **UI 컨트롤**
  - UI 패널 온/오프 토글 버튼 구현

## 폴더 구조

Assets/
├─ EnemyData/ # 적 데이터(ScriptableObject)
├─ Material/ # 적 및 기타 머티리얼
├─ Prefabs/ # 적 프리팹
├─ Scenes/ # 씬 파일
└─ Scripts/ # 주요 스크립트
├─ Character/ # 플레이어, 적 관련 스크립트
├─ Spawner/ # 스폰 관련 스크립트
├─ UI/ # UI 관리 스크립트

yaml
코드 복사

---

## 사용 방법

1. Unity 2022.3.x 버전 이상에서 프로젝트 열기  
2. `SampleScene` 실행  
3. 플레이어는 앞으로 자동 이동하며, 적이 감지되면 자동으로 공격  
4. 몬스터를 처치하면 골드, 경험치 획득 및 퀘스트 진행  
5. UI 버튼으로 버프 및 스탯 업그레이드 가능

## 주요 클래스 및 설명

- `Player`  
  플레이어 이동, 공격, 경험치/레벨 관리

- `Enemy`  
  적의 체력 관리 및 플레이어 공격

- `EnemyData`  
  적의 기본 능력치 저장용 ScriptableObject

- `EnemySpawner`  
  적 소환 담당

- `PlayerSpawner`  
  플레이어 위치 초기화 및 적 소환

- `Buff`  
  버프 효과 및 타이머 UI 관리

- `Condition`  
  체력, 마나, 경험치 UI 및 상태 갱신

- `Quest`  
  퀘스트 진행 및 UI 갱신

- `StatUp`  
  플레이어 스탯 업그레이드 기능
